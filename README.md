# diaryClient

#### 介绍
仿hexo风格博客.

#### 软件架构
采用前后端分离；
客户端使用：vue.js + elementUi
服务端github地址：https://github.com/ygh1483345314/diaryServer

#### 安装教程
1. 安装node.js
2. npm install
3. npm run dev
#### 使用说明
1. 前端页面地址： http://localhost:8080
2. 管理页面登录地址： http://localhost:8080/#/adminLogin
3. 后端api地址：http://localhost:8031/api
